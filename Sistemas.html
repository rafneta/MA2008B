Sistemas LTI continuos y álgebra de bloques

Algunas simplificaciones de bloques, pero se pueden encontrar más en otras fuentes.

Ejemplo 1

Encuentra la simplificación del siguiente diagrama de bloques, es decir, la función de transferencia
syms G_1(s) G_2(s) G_3(s) G_4(s) H_1(s) H_2(s) H_3(s)
F_1 = H_2(s) / G_4(s)
F_2 = G_3*G_4 % serie
F_3 = F_2 / (1-F_2*H_1)
F_4 = G_2 * F_3
F_5 = simplify(F_4 / (1+F_1*F_4)) %simplificamos
F_6 = G_1 * F_5
F_7 = F_6 / (1 + F_6 * H_3);
simplify(F_7)

Ejemplo 2: Motor controlado por armadura

Se supone que no hay perturbaciones
syms k_m R_a L_a J b k_b s
H_1(s) = k_m / (R_a + L_a * s);
H_2(s) = 1 / (J*s + b);
H_3(s) = H_1*H_2
H_4 = H_3 / (1 + H_3*k_b)
H = H_4 * (1/s);
H = collect(H)
