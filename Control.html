Control

Un sistema de control es una interconexión de componentes que forman una configuración del sistema que proporcionará uan respuesta deseada
Un sistema de control en lazo abierto utiliza un regulador o actuador para el proceso directamente sin emplear retroalimentación
Un sistema de control en lazo cerrado usa una medida de la salida y la realimentación de esta señal para compararla con la salida deseada (referencia)
